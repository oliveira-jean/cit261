Objects

An object is a collection of elements; each element can either be a property or a method. A property holds a single data value, and a method will perform some action that is defined as a function.

The easiest way to create an object is by specifying the members through a set of name/value pairs like this:


var myObject = {

color: "Red",

count: 5

};

You can also define methods using this notation, like this:


var myObject = {

color: "Red",

count: 5,

log: function () {

var messageOut = "Quantity: " + this.count + ", Color: " + this.color;

document.getElementById('outputObj1').innerHTML = messageOut;

}

};


Constructors and Instantiation

JavaScript does not use classes. Instead you can reuse an object definition by creating a constructor function.

A constructor can take parameters, and this is often used to initialize the properties. For example, you can write your constructor to take a color and a count parameter:


function Item(color, count){

this.color = color;

this.count = count;

this.msgOut2 = function () {

var messageOut2 = "Quantity: " + this.count + ", Color: " + this.color;

document.getElementById('outputObj2').innerHTML = messageOut2;

};

}


Then create the greenObject variable(Instantition) like this:

var greenObject = new Item("Green", 7);



Prototypes and Inheritance

A prototype is a model or blueprint from which actual objects are instantiated.

You can add a property to an existing object like this:

greenObject.isAvailable = true;

This would add the isAvailable property to the greenObject only; however, if you add this to the Item prototype then the new property would be available for all instances.

You can modify the prototype by accessing it with the prototype property of the constructor function:

Item.prototype.isAvailable = true;

This defines the new property on the prototype and initializes it to true. Now all objects created with the Item() constructor will have the isAvailable property:

var blueObject = new Item("Blue", 3); //Create a new Item object

if (blueObject.isAvailable){

var messageOut3 = "The blue object is available";

}

You can add methods to a prototype in the same way:

Item.prototype.add = function(n){this.count += n;};
