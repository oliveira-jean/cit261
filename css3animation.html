CSS3 - Animation

Animation is process of making shape changes and creating motions with elements.

@Keyframes

Keyframes will control the intermediate animation steps in CSS3. The @keyframe rules is aplyed to the elements that receive the animations.

Examples of @keyframe syntaxe:

@keyframes example1 {

from {background-color: red;}

to {background-color: yellow;}

}


@keyframes example2 {

0% {background-color: red;}

25% {background-color: yellow;}

50% {background-color: blue;}

100% {background-color: green;}

}
